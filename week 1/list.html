Here are few types of lists :-

  1. Unordered lists: list items are displayed using bullet points

    • list item 1
    • list item 2
    • list item 3

  2. Ordered lists: list items are displayed using numerals that can be roman numerals, normal number

    1. list item 1
    2. list item 2
    3. list item 3

  3. Nested listed: this list is an amalgamation of both ordered and unordered lists together

    1. list item 1
      • list item 1
        • list item 1
          1. list item 1
          2. list item 2
          3. list item 3
        • list item 2
        • list item 3
      • list item 2
      • list item 3
    2. list item 2
      1. list item 1
      2. list item 2
      3. list item 3
    3. list item 3
